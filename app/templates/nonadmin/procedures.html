Procedures

Printers Usage

1. Please log in with the registered password

2. Make sure your password matches the name displayed on the monitor

3. Use a USB/Flash drive for printing purposes (PDF file format)

4. Select the "Access Stores Files" menu to view the folders/files on the USB drive that you want to print

5. Choose the "Memory Media" menu

6. Then select the file you want to print

7. Please configure the settings for the file to be printed (paper size, color, page selection as needed) *Errors in color selection may affect the price

8. Once you've finished configuring, select "Start Printing"

9. Please wait for the printed output

10. After completion, please eject the USB used and log out of your account

Scanners Usage

1. Please log in with the registered password

2. Make sure your password matches the name displayed on the monitor

3. Make sure your balance is sufficient before making a photocopy transaction

4. To make a photocopy, please place the document on the ADF (Automatic Document Feeder) until the green light appears or place the document on the glass as needed

5. Then press the start button with the green light

6. Please check the photocopy results to ensure they are accurate

7. If you have completed the photocopy successfully, please log out of your account

Additionals

1. After finishing using the printer, don't forget to log out

2. Do not leave any items on the photocopy machine (used paper, USB drives, etc.)

3. If you encounter any damage or issues, please inform the library staff/SEP

4. Mistakes in printing or copying documents are your responsibility

Book Borrowing

1. Students may select the books they wish to borrow

2. Bring the selected book(s) to the library staff/SEP for scanning

3. Inquire with the library staff about the due date for returning the book(s)

Book Returning

1. Students may return the borrowed book(s) to the library personnel

2. Ensure that you return the book(s) before the designated return deadline

3. If a book is returned after the due date, a fine will be imposed in accordance with the library's policy
